# Style Canvas — Fashion E-Commerce Store

A modern fashion e-commerce website where users can browse and shop premium clothing across multiple categories. Built with Next.js and deployed on Vercel.

## Live Demo

[style-canvas-inky.vercel.app](https://style-canvas-inky.vercel.app)

## Features

- Shop by category — Shirts, Jeans, Sneakers, Jackets
- Shopping cart
- About, Contact, FAQ, and Shipping pages
- Clean, responsive design

## Tech Stack

- Next.js
- JavaScript
- Tailwind CSS

## Getting Started

```bash
git clone https://github.com/Mehdi-Chandia/StyleCanvas.git
cd StyleCanvas
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
